# Insurance & Patient Management System

A desktop-based insurance and patient management application built using **Python (Tkinter)** and **MySQL**.  
This project was developed as part of academic learning to understand database-driven applications and GUI development.

## Features
- Patient registration, update, and deletion
- Medical bill management (add, edit, delete)
- Doctor details linked to patients
- Insurance policy and claim management
- Relational data handling using MySQL
- Scrollable and form-based Tkinter UI

## Tech Stack
- Python
- Tkinter (GUI)
- MySQL
- mysql-connector-python

## Database
The project uses a MySQL database with multiple related tables:
- patient
- medbill
- doctor
- policy
- insurance
- claim
